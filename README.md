# QQ Mail

`qq-mail` 是一个用于 Codex 的本地 QQ 邮箱插件仓库。

## 内容

- `mcp/qq-mail-mcp/server.py`：本地 MCP 服务端实现
- `mcp/qq-mail-mcp/scripts/smoke_mcp.py`：MCP 冒烟验证脚本
- `skills/qq-mail/SKILL.md`：Codex skill 说明
- `.codex-plugin/plugin.json`：插件清单
- `.mcp.json`：MCP 配置

## 用途

用于在 Codex 中通过 QQ Mail 执行邮件相关操作，例如：

- 查看收件箱
- 搜索邮件
- 发送邮件
- 标记已读或未读
- 移动、复制、删除邮件

## 说明

这是一个面向本地 Codex 环境的插件仓库，具体运行方式以仓库内的 skill 和 MCP 配置为准。
